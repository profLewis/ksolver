Kernel Solver¶

. UCL and NERC NCEO.

Regularised kernels solver

Fits the model of reflectance \(\rho\):

\[ \hat\rho = f_0 \left[ 1 + f_1' k_1(\Omega,\Omega') + f_2' k_2(\Omega,\Omega') \right] \]

with \(f_1' = f_1/f_0\) and \(f_2' = f_2/f_0\) constant over time and \(f_0\) regularised (using GCV to determine smoothness).

One core code is a fast smoother with regularisation from Garcia (2010) implemented in Python as the function smoothn.

Another is the kernel BRDF model code in the class Kernels.

Garcia, D. (2010), Robust smoothing of gridded data in one and higher dimensions with missing values, Computational Statistics and Data Analysis 54 (2010) 11671178
